# 📊 Student Performance & Traffic Statistical Analysis

An end-to-end Data Science project that explores and analyzes **Student Academic Performance** and **Traffic Volume Patterns** using statistical techniques, Exploratory Data Analysis (EDA), and interactive visualizations.

The project includes:

* 📈 Comprehensive Exploratory Data Analysis (EDA)
* 📊 Statistical Summary & Insights
* 🔍 Correlation and Trend Analysis
* 🚦 Traffic Volume Pattern Analysis
* 🎓 Student Performance Evaluation
* 🌐 Interactive Streamlit Dashboard
* 🤗 Hugging Face Spaces Deployment Support

---

## 🚀 Features

### Student Performance Analysis

* Data Cleaning & Preprocessing
* Descriptive Statistics
* Correlation Analysis
* Subject-wise Performance Analysis

### Traffic Volume Analysis

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-performance-traffic-analysis.git

cd student-performance-traffic-analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Jupyter Notebook

```bash
jupyter notebook notebooks/eda.ipynb
```

---

## 🌐 Run Streamlit App

```bash
streamlit run app.py
```

The application will be available at:

```bash
http://localhost:8501
```
